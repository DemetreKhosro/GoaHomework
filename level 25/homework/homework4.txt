inline CSS - ვიყენებთ style ატრიბუტს ყველა იმ ელემენტში ცალ-ცალკე რომლის გასტილვაც გვინდა
Internal CSS - ვაჯგუფებთ რამდენიმე ელემენტს style თეგში და ვსტილავთ მათ css დეკლარაციით.
მე პერსონალურად Internal css-ს ვანიჭებ უპირატესობას რადგან ის კოდში უფრო ადვილი წასაკითხი და დალაგებულია, ხოლო inline css კოდს ძნელად წასაკითხს ხდის (იმ შემთხვევაში თუ ძალიან ბევრ ელემენტს გავსტილავთ.)
