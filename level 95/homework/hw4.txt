state - მდგომარეობა - ინახავს მონაცემებს და ეკრანის განახლებას აიძულებს მათი განახლებისას

hook - კაუჭა - ფუნცქია რომელიც საშუალებას გვაძლევს გამოვიყენოთ React-ის სხვადასხვა ფუნქციები როგორიც არის hook-ები

built-in hook - React-ში ჩაშენებული კაუჭები მაგალითად useState/useEffect

useState - კაუჭა რომელიც კომპონეტში ქმინს ცვლადს მონაცემებისს შესანახად და მის შესაცვლელ ანუ setter ფუნქციას

setter function - useState-დან მიღებული ფუნქცია რომლის მოვალეობაა state-ის მნიშვნელობის განახლება

component re-render - როცა react თავიდან კომპონენტის ფუნქციას და ანახლებს ეკრანს როცა state განახლდება
